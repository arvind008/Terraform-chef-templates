# Templates for provisioning applications in private and public cloud
This repository consists of templates for provisioning application. These templates are meant to be used from deepinsight. Nothing prevents users from directly using cloud formation tools or Terraform application to  use these templates.


## Template catalogue
Catalogue of templates can be generated by executing the command, 

```commandline
python buildtemplateinfo.py
```

The generated template catalogue is for the consumption of deepinsight application. Do not modify this application. Do not commit generated catalogue to repository.


## Guidelines for committing new template.
 - Every template should be comminted to a new directory.
 - Top level of template directory should have a file template_info.josn. Below is a sample template_info.json
```
{
  "name": "ElasticSearch",
  "type": "terraform",
  "target_cloud": "vcenter",
  "description": "Create a Elastic Search cluster using terraform and ansible elastic role",
  "prerequisites": "",
  "limitations": ""
}

```

## Sample parameters.json

- Input Types: string, integer, float, list ( single / multi select )
- hidden: use this property of input configuration, if we want to pass default value implicitly without displaying it in UI ( like refresh interval, default credentials.. )
- label, name, type are required configuration properties
- tooltip, placeholder, value are optional configuration properties



```json
{
                label: 'Node Count',
                fieldName: 'node_count',
                dataType: 'integer',
                min: 4,
                max: 10,
                defaultValue: 5,
                placeholder: 'number of nodes to create for data',
                tooltip: 'number of nodes to create for data'
},
{
                label: 'User Name',
                fieldName: 'username',
                dataType: 'string',
                defaultValue: 'root',
                placeholder: 'username to be used for authentication',
                tooltip: 'username to be used for authentication',
                hidden: true
},
{
                label: 'Password',
                fieldName: 'password',
                dataType: 'password',
                defaultValue: 'DeepInsight@123',
                placeholder: 'Password to be used for authentication',
                tooltip: 'Password to be used for authentication',
                hidden: true
},
{
                label: 'Instance Type',
                fieldName: 'instance_type',
                dataType: 'list',
                multiselect: true,
                options: [
                      {
                           label: 'Small',
                           value: 't2.small'
                      },
                      {
                           label: 'Medium',
                           value: 't2.medium'
                      }
                ]
                defaultValue: 't2.small',
                tooltip: 'aws instance type for all nodes'
}

```
